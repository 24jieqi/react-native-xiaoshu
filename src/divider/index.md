# Divider 分割线

> 用于将内容分隔为多个区域。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割。

React Native 在 iOS 端的 borderStyle 暂时不支持虚线，使用 SVG 的方式绘制。
