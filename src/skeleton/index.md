# Skeleton 骨架屏

> 在需要等待加载内容的位置提供一个占位图形组合。

## 何时使用

- 网络较慢，需要长时间等待加载处理的情况下。
- 只在第一次加载数据的时候使用。
- 图文信息内容较多的列表/卡片中。
