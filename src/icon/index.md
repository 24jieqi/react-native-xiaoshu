# Icon 图标

> SVG 图标

24\*24 大小图标，默认触摸、点击尺寸大小是 44\*44，可通过 `hitSlop` 自定义向外扩展触摸、点击的范围。可添加 `onPress` 点击事件，如果没有图标将事件冒泡到父级， `pointerEvents="none"`。
